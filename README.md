# DROPLINE – Men’s Apparel E-Commerce Store

## Overview
**DROPLINE** is a modern, frontend-focused e-commerce web application showcasing **men’s streetwear apparel**, including **tees, hoodies, sweatshirts, jackets, and drop shoulder styles**.

All products displayed in the store are **dynamically fetched from an API**, simulating a real-world online shopping experience.  
This project is built for **learning, practice, and portfolio purposes**.

---

## Features
- 🧥 Men’s apparel collection (tees, hoodies, sweatshirts, jackets)
- 📦 Products dynamically **fetched from an API**
- 🛒 Add-to-cart functionality (frontend-based)
- ➕➖ Quantity increase / decrease
- ❌ Remove items from cart
- 🧾 Cart sidebar for better user experience
- 🌙 Light / Dark mode toggle
- 📱 Fully responsive design

---

## Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Bootstrap / Tailwind CSS**
- **API:** JSON-based product API (mock or public API)
- **Icons:** Font Awesome

---

## How It Works
- Product data is fetched using the **Fetch API**
- JavaScript dynamically renders product cards
- Cart state is managed on the frontend
- UI updates in real time when items are added or removed
